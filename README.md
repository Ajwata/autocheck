# Підбір та перевірка авто під ключ

Сучасний сайт для компанії, що спеціалізується на професійному підборі та комплексній перевірці автомобілів у Львові та області.

## 🎯 Про нас

Безпечна купівля автомобіля без ризиків і прихованих дефектів. Наша компанія спеціалізується на:

- **Підборі авто під ключ** - повний супровід від першого запиту до покупки
- **Перевірці авто на виїзд** - комплексна перевірка з професійним обладнанням
- **Ендоскопії двигуна** - виявлення прихованих дефектів
- **Діагностиці авто** - перевірка електронних систем

## 🚀 Розгортування на GitHub Pages

1. Завантажте файли в GitHub репозиторій
2. Перейдіть в Settings → Pages
3. Оберіть гілку `main` / `master` як джерело
4. Сайт буде доступний за `https://your-username.github.io/repository-name`
